IGVC Perception Dashboard

Reference page for the current camera and lane-perception setup. The final target is front + left + right. The back camera is intentionally out of scope. This page is the single place to check live camera streams, lane segmentation, and current camera status.

Stream Configuration

Use the Jetson web stream host here. For a GitHub-hosted HTTPS page, embedded live video only works cleanly if this base URL is also HTTPS.

Switch to links only if your browser blocks mixed HTTP content.

Current Assumptions

  • Primary competition lane following uses the front camera first.
  • Left camera is available as a live supporting view.
  • Right camera is enabled in software but not currently detected by the SDK.
  • Back camera is not part of the final product.
  • This dashboard is for quick reference. Heavy debug tools should still stay off in race mode.
